4.14.3 Physics using ODE

The ‘odedynam’ plugin implements physics for Crystal Space using the ODE library.

It is not advised anymore to use the ODE plugin, in most cases, the Bullet plugin will be more interesting.

Stepping the Physical Simulation

To make sure the physics simulation proceeds correctly in time you must tell ODE how much time has elapsed since previous frame so that it can calculate the collisions and physical interactions that happened during that time. The iDynamics->Step() function is responsible for that. As a parameter it has a delta which is a floating point number that represents the number of seconds to calculate. The ODE plugin will automatically make sure that ODE simulation is called in fixed time steps. This increases stability of the physics simulation.
