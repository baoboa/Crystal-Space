5.8 Making Levels and Models in Blender

Blender is one of the most popular open source 3D modelers available and can be used to create models and scenes for Crystal Space. You can find Blender here:

http://www.blender3d.org

To export your objects into Crystal Space, you will need to use the io_scene_cs add-on script. This script is valid for the Blender 2.6x series and above.

If you want to use the Blender 2.4x series, then you should use the blender2crystal script instead (available from http://leapingcat.org/blender2crystal)

Another alternative solution, allowing to load directly the Blender files within Crystal Space without the need to export them previously, can be found with the Assimp's generic import of models, although it produces generally less good results for Blender files.

The io_scene_cs add-on script is still in Beta stage, although it works quite well with most objects and scenes. It can export:

  • static meshes into Crystal Space Genmesh meshes
  • animated meshes, along with their skeleton, morph targets, bone influences, animations and sockets, into Crystal Space Animesh meshes
  • materials and textures
  • scenes with portals and multiple sectors
  • groups of objects as instanced meshes
