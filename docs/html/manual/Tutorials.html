4.2 Tutorials

This chapter contains various tutorial to help you quickly familiarize yourself with the various aspects of the Crystal Space development environment.

Note, these tutorials only talk about the coding issues when working on Crystal Space applications. They don't talk about how to build the tutorials as that is specific to the operating system and compiler that you use. Here is a list of where you should go in the manual to find information about this:

  • For MSVC 7 or higher (see section Creating an External MSVC 8, 9 or 10 Application).
  • Using Dev-CPP (see section Creating an External Crystal Space Application Using Dev-CPP).
  • Using GCC based compiler (MingW or GNU/Linux based) (see section Creating an External Crystal Space Application).
