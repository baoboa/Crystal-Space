4.9.8.4 Sector Structure

In this subsection we go deeper into the structure of a sector. A very common case to use sectors is to define a sector with one GenMesh which is convex and makes up the outer hull of the sector. Sectors don't have to be defined like this but this is very common. You can then make a maze like structure with convex rooms connected to other convex rooms with portals. Non-convex attributes of the rooms can be added as separate detail objects.
