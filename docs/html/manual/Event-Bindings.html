4.4.5 Bindings

Syntax for keys in 'cooked' mode

 
[[modifier]+|-<key>

Example:
Shift+Up
Shift-Up
Note: + - is the same thing

Syntax for Mouse

 
[[mouse number]Mouse<Axis|Button><number>

Example:
MouseAxis0
0MouseAxis0
1MouseButton1

Note: first and second examples are the same (0 is default).

Syntax for Joystick

 
[[joy number]Joystick<Axis|Button><number>

Example:
JoystickAxis0
0JoystickAxis0
1JoystickButton1

Note: first and second examples are the same (0 is default).
