4.22.5 Various Tools

BugPlug can do a lot more. Here is a summary of some of the more useful features:

  • Colorize sectors (‘ctrl-d ctrl-shift-c’): with this feature BugPlug will give every sector a random color. That way when you walk around your world it is easy to see where a new sector starts.
  • Dump engine (‘ctrl-d d’): dump all materials, meshes, sectors, factories, and lights that are loaded. This can be a useful tool to see what the engine currently contains. Note that it produces a lot of output.
  • Dump camera (‘ctrl-d shift-c’): dump the contents of the current camera (orientation, position, and FOV).
  • Change FOV (‘ctrl-d f’): change the Field Of Vision setting.
  • List plugins (‘ctrl-d ctrl-alt-l’): list all currently loaded plugins.
  • Click mesh (‘ctrl-s right-mouse-button’): this doesn't select a mesh but instead it prints out the mesh that was hit and the distance between the camera and that mesh.
