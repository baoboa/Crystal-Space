2.4.3 The “cs-winlibs” Packages

Rather than having to manually install the external libraries needed for Crystal Space, Microsoft Visual C++ 8/9/10, MinGW/MSYS, and Cygwin users can download a single package which contains all common headers, pre-built library files, and dynamic link libraries in pre-compiled form, all in versions known to work with this particular release of Crystal Space.

You can obtain it from the following location:

32-bit version

http://www.crystalspace3d.org/downloads/cs-winlibs/cs-win32libs-2.1_002.exe

64-bit version

http://www.crystalspace3d.org/downloads/cs-winlibs/cs-winlibs-x64-2.1_002.exe
