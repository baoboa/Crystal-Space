2.4.7.1 Library Prerequisites

In addition to obtaining the project source code (see section Where to Get Crystal Space), you may also want to download and compile several support libraries used for image manipulation (see section External Libraries). The libraries ‘libpng’ and ‘libjpeg’, which are present on many Unix installations, are absent from MacOS/X. The optional ‘libmng’ is less common.

Various package managers exist to simplify the task of installing command-line tools and development libraries, such as ‘libpng’ and ‘libjpeg’. Choose one of the package managers below and follow its documentation to install prerequisites libraries.

  • MacPorts
  • Fink
  • Homebrew

Alternately, the libraries can be built manually. You can download the required external library source code packages from their canonical download sites (see section External Libraries). Follow the included documentation to build and install each library.
