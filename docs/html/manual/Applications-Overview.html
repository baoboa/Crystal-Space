3.2 Applications Overview

This section provides a quick overview of the various applications which are bundled with Crystal Space. In some cases, more detailed information for a particular application may be found in subsequent sections.

Most of these applications provide a basic description on how to use them by launching them along with the ‘--help’ parameter.

3ds2lev

A 3DS to Crystal Space map converter (work in progress!).

sndtest

OpenAL sound test.

avatartest

An application to test and demonstrate the animated mesh animations.

basemapgen

Base map generator for terrain2. For a given terrain, it will generate the global texture to be applied on it by combining the textures splatted on the terrain.

ceguitest

A test application for the Crystal Space Crazy Eddie's GUI System plugin.

collada2cs

A COLLADA to Crystal Space converter.

consoletest

A Crystal Space console output testing application.

csbench

A Crystal Space benchmarking application.

csdemo

A demonstration of a medieval First Person Shooter showing several features of Crystal Space

csmocapviewer

A viewer for motion captured data

csfgen

A TrueType to Crystal Space font converter.

csimagetool

The Crystal Space image manipulator.

distfieldgen

Distance field generator for images (see here for more information on the ways it can be used).

docconv

An utility to convert between document formats. For the conversion to and from binary XML, See section Binary XML.

eventtest

An application for the testing of the event system with keyboard, mouse and joystick.

g2dtest

The Crystal Space canvas plugin test application.

genmeshify

A tool to convert from the old thing mesh to a genmesh.

heightmapgen

todo

heightmapproc

todo

imptest

A test application for imposters.

isotest

An application demonstrating how you can use the 3D engine to simulate a semi-isometric view.

jobtest

todo

joytest

A test application on the use of joysticks.

levtool

todo

lghtngtest

todo

lighter2

Tool to compute static lighting for a CrystalSpace world. See section Using lighter2.

maya2spr

A tool to convert models from Maya into Crystal Space.

mazing

A tutorial application. See section Mazing Tutorial: Small Game.

md32spr

todo

mdl2spr

This application is a Quake ‘.mdl’ and ‘.md2’ model converter. It converts model files into native Crystal Space sprites. See section Quake Model Converter (mdl2spr).

optimisedata

todo

partconv

todo

pathtut

A demonstration on the way to use the sequencing system. See section Sequence Manager.

phystut

An application to test and demonstrate the use of the physics plugin. See section Collision Detection and Physics.

pysimp

An extremely simple example demonstrating use of the Python plug-in module.

shadertut

A tutorial on the usage of shaders.

shagnetron

todo

simdtest

Tests for the Single Instruction on Multiple Data available on your system.

simpcd

A demonstration on the way to use collision detection. See section Basic Collision Detection.

simple1

A tutorial application. See section Simple Tutorial 1: Basic Setup, World Creation. The code of this application is a good candidate if you want to test some piece of code.

simple2

A tutorial application. See section Simple Tutorial 2: Adding a Mesh. The code of this application is a good candidate if you want to test some piece of code.

simplept

A tutorial for procedural textures. See section Procedural Texture System.

simpmap

A tutorial application. See section Simple Tutorial 3: Map Loading, Collision Detection.

simpvs

todo

sndtest

An application for testing the Crystal Space sndsys sound system.

startme

The main demonstration application, providing access to many Crystal Space's demos.

threadtest

A test application on the threaded loader.

tri3dtest

A test application on the automatic triangulation of a mesh.

unittest

todo

viewmesh

A tool to preview Crystal Space models and libraries. See section Previewing Models (viewmesh).

vsh

This is the test application for the Virtual File System (VFS). It works very similar to the standard Unix shell programs. See section Virtual File System (VFS).

walktest

The quintessential Crystal Space test program. ‘walktest’ is the test harness for most of the features which get added to the engine. It is not necessarily a good programming example, but if you are interested in seeing an engine feature in action, this is probably the place to look. See section Walktest, the Basic Test Application.

waterdemo

A demonstration application, showing how to modify dynamically a material to achieve a water animation.

waterdemo2

A demonstration application about the 'ocean' watermesh. See section Watermesh Mesh Object.

wxtest

A demonstration application showing how to use the iWxWindow plugin.
