1.4 Using Crystal Space For Your Project

What if you want to use Crystal Space in your project? The best approach at this point in time is to become a member of the mailing lists and listen to how everything develops. If you're really serious about things, you can join discussions and help develop for the engine. Jorrit and other core developers will gladly add contributed code to Crystal Space, provided we can use the code.

Note that you also need to agree to the LGPL details, but it shouldn't be a real problem for OSS developers. See section Licenses.

We are always interested in hearing what other people are doing with Crystal Space. Even if you only use Crystal Space for small private projects, we would like to hear about it, just to satisfy our curiosity.
